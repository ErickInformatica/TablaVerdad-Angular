Generador de Tabla de Verdad

Para ingresar una preposicion puede utilizar los siguientes:

  • & = Conjuntor (y)
  • | = Disyuntor (o, v)
  • => = Condicionador (→, si... entonces...)
  • <=> = Bicondicionador (↔, Si y solo si...)
  • ! = Negacion (~, no)
Puede usar parentesis "()", si lo necesita. Para comparar multiples preposiciones usar coma (,).
